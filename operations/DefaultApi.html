calcPrecoDataGet

URL

http://ws.correios.com.br/calculador/CalcPrecoPrazo.asmx/CalcPrecoData

HTTP Method

GET

Response Type

Parameters

  • Query: nCdEmpresa String

    Seu código administrativo junto à ECT. O código está disponível no corpo do contrato firmado com os Correios. ##### Obs: O parâmetro tem que ser passado mesmo vazio.

  • Query: sDsSenha String

    Senha para acesso ao serviço, associada ao seu código administrativo. A senha inicial corresponde aos 8 primeiros dígitos do CNPJ informado no contrato. A qualquer momento, é possível alterar a senha no endereço ##### Obs: O parâmetro tem que ser passado mesmo vazio.

  • Query: nCdServico String

    | Código | Serviço | |--------|--------------------------------------| | 04014 | SEDEX a Vista | | 04065 | SEDEX à vista pagamento na entrega | | 04510 | PAC à vista | | 04707 | PAC à vista pagamento na entrega | | 40169 | SEDEX 12 | | 40215 | SEDEX 10 (à vista e a faturar) | | 40290 | SEDEX Hoje Varejo | ##### Obs: Pode ser mais de um numa consulta separados por vírgula.

  • Query: sCepOrigem String

    CEP de origem hífen. Ex: 88102080

  • Query: sCepDestino String

    CEP de destino sem hífen. Ex: 05311900

  • Query: nVlPeso String

    Peso da encomenda, incluindo sua embalagem. O peso deve ser informado em quilogramas. Se o formato for Envelope, o valor máximo permitido será 1 kg

  • Query: nCdFormato Integer

    Formato da encomenda (incluindo embalagem). Valores possíveis: 1. Formato caixa/pacote 2. Formato rolo/prisma 3. Envelope

  • Query: nVlComprimento String

    Comprimento da encomenda (incluindo embalagem), em *centímetros*

  • Query: nVlAltura String

    Altura da encomenda (incluindo embalagem), em *centímetros*. Se o formato for envelope, informar zero (0).

  • Query: nVlLargura String

    Largura da encomenda (incluindo embalagem), em *centímetros*.

  • Query: nVlDiametro String

    Diâmetro da encomenda (incluindo embalagem), em *centímetros*.

  • Query: sCdMaoPropria String

    \"Indica se a encomenda será entregue com o serviço adicional mão própria. Valores possíveis: S ou N (S – Sim, N – Não)\"

  • Query: nVlValorDeclarado String

    Indica se a encomenda será entregue com o serviço adicional valor declarado. Neste campo deve ser apresentado o valor declarado desejado, em Reais

  • Query: sCdAvisoRecebimento String

    \"Indica se a encomenda será entregue com o serviço adicional aviso de recebimento. Valores possíveis: S ou N (S – Sim, N – Não)\"

  • Query: sDtCalculo date

    Data tipo dd/mm/aaaa. Ex: 21/10/2017
